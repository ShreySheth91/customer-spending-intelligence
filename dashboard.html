Fraud rate by age group
Older cardholders (55+) show higher fraud exposure and higher avg transaction size — likely higher-value targets. Age is a useful continuous feature alongside the discretized group.
Geographic distance signal
Median distance is nearly identical — Legit: 78.2 km vs Fraud: 77.9 km. Distance is a weak standalone signal; combine with hour + amount for compound features.

07 · what's next
Phase 2
RFM Segmentation

K-Means clustering on the customer_aggregated.csv table to identify high-value, at-risk, dormant, and champion segments. Silhouette scoring for optimal k.

Phase 3
Fraud / Churn Modeling

XGBoost classifier with SMOTE resampling. Primary metric: PR-AUC. SHAP values for interpretability. CLV regression as secondary output.

Phase 4
Interactive Dashboard

Streamlit or Dash app with segment explorer, real-time fraud scoring interface, and model explanation panel. Deployable to Hugging Face Spaces.
